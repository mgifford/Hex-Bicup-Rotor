VINDSNURR

HBR-6-14V · Rev 1.1 · Open source · MIT licence

VINDSNURR is an open-source vertical-axis drag wind turbine based on a 14-vertex hexagonal bicupola frame. Six triangular sail panels mounted on a fully triangulated rod-and-hub structure drive a central shaft in a Savonius-style drag cycle. The frame is designed for 3D-printed hubs, fiberglass or carbon rod struts, and ripstop nylon sails.

Status: Design phase. OpenSCAD files are complete and ready to print. No physical prototype has been built or tested. All performance figures are theoretical estimates.

Design files

OpenSCAD source files for 3D printing are in the GitHub repository under 3d/:

  • vindsnurr_params.scad — shared parameters and geometry constants (the single source of truth)
  • hub_apex.scad — Hub T and Hub B, degree-6 with 6204-2RS bearing seat. Print 2.
  • hub_ring.scad — Hub U1–U6 and L1–L6, degree-5. Print 12. Same part, L hubs flip 180°.
  • hub_test_jig.scad — socket fit and angle calibration. Print this first.

Open any .scad file in OpenSCAD (free), press F6 to render, then export as STL.

Design inspiration

VINDSNURR was inspired by the GeoWind GW1200, a geodesic VAWT using an icosahedron frame that won the CES 2026 Innovation Award and iF Design Award. Both designs apply the same thesis — a fully triangulated polyhedral frame serves simultaneously as structure and aerodynamic geometry — but use different polyhedra. See the geometry comparison for the full analysis.
